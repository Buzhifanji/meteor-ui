# button
## 文档地址
- [button-type](https://developer.mozilla.org/zh-CN/docs/Web/HTML/Element/button)

## 属性

### attr-type（按钮的 DOM 的 type 属性）
- button（默认）
- submit
- reset

### type（按钮类型）
- default（默认）
- primary
- success
- info
- warning
- error
### disabled(设置按钮是否禁用)
- true
- false（默认）
### loading(设置按钮载入状态)
- true
- false（默认）
### size(设置按钮大小)
- tiny
- small
- medium（默认）
- large
### block(设置按钮为块级元素)
- true
- false（默认）
## 事件
- click（点击按钮时的回调）
